# Python Classes (Object-Oriented Programming)

This directory contains my work on **Object-Oriented Programming (OOP)** as part of **Python Fundamentals**.
The focus of this section is on building a **strong conceptual and practical understanding of OOP**, including how real-world Python code is structured, reused, and extended.

---

## Concepts Covered

- Defining and using classes
- Instance attributes and methods
- Class inheritance
- Method overriding
- Composition (using objects as attributes)
- Modular class design
- Importing classes across files and packages

---

## Learning Objective

The goal of this section is to move beyond writing isolated scripts and toward:
- Thinking in terms of **objects and responsibilities**
- Writing **maintainable and reusable code**
- Structuring Python code the way it is done in real projects
