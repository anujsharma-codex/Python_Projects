# Practice Questions – Object-Oriented Programming (Python)

This folder contains practice problems and solutions focused on
**Object-Oriented Programming (OOP)** in Python.
The goal of this folder is to **build clarity and confidence** in OOP concepts
by solving structured questions and experimenting with code.

---

## Topics Included

### 1. Class and Object Basics
- Defining classes
- Creating objects
- Using the `__init__` method
- Instance attributes and methods

---

### 2. Working with Classes and Instances
- Reading and modifying attributes
- Writing methods to update object state
- Passing data between methods
- Understanding how instances behave independently

---

### 3. Inheritance
- Parent and child classes
- Using `super()`
- Extending and overriding behavior
- Modeling real-world relationships with inheritance

---

### 4. Importing Classes
- Splitting classes into separate files
- Importing classes across modules
- Understanding package structure
- Writing cleaner, reusable OOP code

---

## Purpose

- Strengthen OOP fundamentals
- Learn how real Python projects are structured
- Prepare for larger, modular applications

---

## Notes

- Code is learning-oriented, not production-optimized
- Some files may include iterative or exploratory solutions
