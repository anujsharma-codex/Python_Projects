# Python Modules (OOP Support Code)

This directory contains **reusable Python modules** built while learning and practicing **Object-Oriented Programming (OOP)**.
Each module defines one or more related classes that are imported and used across notebooks and scripts, following real-world Python project structure.

---

## Purpose of This Folder

- Separate **class definitions** from execution logic
- Encourage **code reuse** instead of copy-paste
- Practice Python’s **import system** and package structure
- Simulate how production-level Python projects are organized

---

## Design Philosophy

- Each module has a **clear responsibility**
- Related classes may live together when logically connected
- Modules are written to be **import-safe** (no execution on import)
- Focus is on clarity, not over-engineering

---

## Key Learnings

- Creating Python packages using `__init__.py`
- Absolute imports across modules
- Managing dependencies between parent and child classes
- Avoiding circular imports
- Structuring OOP code for scalability
