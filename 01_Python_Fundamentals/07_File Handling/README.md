# File Handling – Fundamentals

## Overview

This folder contains beginner-level exercises focused on mastering file handling in Python.

The objective is to understand how Python interacts with text files using different techniques such as:

- Reading entire files
- Looping through file objects
- Storing lines in lists
- Writing user input to files
- Appending data safely
- Replacing text dynamically
- These exercises build the foundation for real-world tasks like:
- Logging systems
- Data storage
- Report generation
- Data preprocessing for analytics

## Exercises Implemented

### 1️⃣ Learning Python (learning_python.txt)

Concepts Covered:

- Opening files using with
- Reading entire file using .read()
- Iterating over file objects
- Using .readlines()
- Working with file content outside the with block

#### Goal:

Understand different approaches to reading file content and when each is useful.

### 2️⃣ Learning C (Text Replacement)

Concepts Covered:

1. String replacement using .replace()
2. Reading line-by-line processing
3. Dynamic text modification

#### Goal:

Simulate real-world text preprocessing where content must be cleaned or modified before use.

### 3️⃣ Guest (guest_book.txt)

Concepts Covered:

1. Writing to files using .write()
2. Capturing user input
3. Creating files dynamically

### Goal:

Understand basic file creation and data persistence.

## 4️⃣ Guest Book (guest_book.txt)

Concepts Covered:

1. while loops with file writing
2. Appending data using "a" mode
3. Ensuring each entry is stored on a new line

### Goal:

Simulate a real visitor logging system.

## 5️⃣ Programming Poll (like_programming_reasons.txt)

Concepts Covered:

1. Continuous input collection
2. Appending multiple responses
3. Storing survey-like data

### Goal:

Build a simple data collection system similar to real-world survey storage.

## Skills Strengthened:

- File modes: r, w, a
- Context managers (with)
- Safe file handling practices
- Input/output flow
- Data persistence basics
- Clean separation of logic and file operations

## Why This Matters

### File handling is foundational for:

- Working with CSV datasets
- Logging ML experiments
- Storing intermediate data
- Preprocessing raw text data
- Handling configuration files
- Backend development

### Without strong file handling fundamentals, moving into:

- Pandas
- NumPy
- SQL
- Data engineering
- Machine learning pipelines
- becomes harder than it should be.
