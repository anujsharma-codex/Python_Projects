# Pets Information Collector

## Description

This Python program allows users to enter pet details interactively and stores them in a structured format using a list of dictionaries.
Each pet is stored as a dictionary containing:

- Pet name
- Animal type
- Owner’s name

All pet dictionaries are stored in a list named pets.
After the user finishes entering data, the program displays all stored pet information in a clean, formatted output.

## Learning Objectives

- Work with lists and dictionaries together
- Handle user input in loops
- Organize code using functions
- Display structured data clearly

## Concepts Used

- Python functions
- Lists
- Dictionaries
- while loop
- for loop with enumerate
- String formatting using format()
- String methods: .strip() and .title()

## Program Flow

1. Define a function to collect pet details from the user
2. Store each pet as a dictionary
3. Append each dictionary to the pets list
4. Ask the user whether to continue
5. After input ends, loop through the list
6. Print all pet details with numbering
